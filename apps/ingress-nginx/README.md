# ingress-nginx

ingress-nginx is an Ingress controller for Kubernetes using NGINX as a reverse proxy and load balancer.

source: [GitHub](https://github.com/kubernetes/ingress-nginx)

> ⚠️ **Important:** `ingress-nginx` will be replaced by **Envoy Gateway** as part of the migration from the Kubernetes **Ingress API** to the **Gateway API**.
