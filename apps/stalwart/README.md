# stalwart

Stalwart is an open-source mail and collaboration server designed for the modern internet. Stalwart supports a wide range of protocols including JMAP, IMAP4, POP3, SMTP, CalDAV, CardDAV, and WebDAV, covering email, calendars, contacts, and file storage. Built in Rust, Stalwart is designed to be secure, fast, reliable, and scalable, and is capable of running everything from small personal mail servers to large, distributed enterprise deployments.

Source: [GitHub](https://github.com/stalwartlabs/stalwart)
