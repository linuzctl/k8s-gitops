# promtail

Promtail is an agent which ships the contents of local logs to a private Grafana Loki instance or Grafana Cloud.

source: [GitHub](https://github.com/prometheus-community/helm-charts/tree/main/charts/kube-prometheus-stack)

> [Promtail is now deprecated](https://grafana.com/docs/loki/latest/send-data/promtail/) and will enter into Long-Term Support (LTS) beginning Feb. 13, 2025. Migration is planned to Alloy.
